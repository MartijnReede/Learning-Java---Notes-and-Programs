Goals for this program:

Making an email address creator for the company Patagonia with the functions below: 
	- Generate an email address with: firstname.lastname@department.patagonia.com.
	- User must set a department.
	- Program generates password.
	- Possibility to set inbox capacity in mb.
	- Possibility for user to change the email address.  
	- Possibility for user to change the password. 
	- Option to exit the program and set email information permanently.

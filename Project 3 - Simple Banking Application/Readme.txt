Goals for this program: 

Making a simple bankapplication with the functions below:
	- Main menu for the account holder to choose an option from.
	- Display name account holder and personal ID.
	- Option to view current balance.
	- Option to withdraw money, if balance amount is greater than requested withdrawal.
	- Option to deposit money.
	- Option to see the last transaction. If the user made one before. 
	- Option to exit the program
